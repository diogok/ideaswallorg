About

This service and applications are developed by Diogo Souza da Silva, a 20 years old Brasilian student.

It was developing with the intention to be able to record his own ideas wherever he is, and to learn the technologies related.

The lightbulb icon/image used in the mobile version or the app icon is from Liam McKay , the website images are from Oxygen Icons and the Desktop interface icons are from Crystal SVG Icons.

This service was developed following RESTful principles for the webservice, it runs on google app engine cloud computing java enviroment, is written in PHP running on top of Quercus and storing using google datastorage database using Java JDO objects.

The web interface uses assyncronous request to provide a rich experience, using the JQuery framework.

The Desktop application is build using the latest JavaFX technology, SUN's newest techonlogy for RIA, and the Mobile version is build on Java Mobile plataform, J2ME, that is present on lot's of devices .

The developing enviroment is a Debian GNU/Linux, running the LAMP stack and the Jetty server with Quercus, using VIM as editor for the PHP, HTML, CSS and JavaScript, and Netbeans for the Java technologies.

There are still lot's of improvements todo, specially on the interfaces but also features to add to the service. Here is what is planned to be implemented soon:

  • Offline function and the abilit to synchronize over the web ondemand.
  • Possibilit to share some tags with other users or general public tags.
  • Integration with other services, like google calendar.
